Sobre mim

Desde a adolescência, sou apaixonado por programação, explorando e aprendendo várias linguagens de desenvolvimento, tanto para front-end quanto para back-end. Tenho dedicado bastante tempo ao estudo autodidata.

  • No front-end, aprendi HTML, CSS e um pouco de Angular, criando algumas páginas web interativas.
  • No back-end, fiz projetos simples usando Node.js e Django, aprendendo a criar a lógica de funcionamento de aplicações.
  • Também usei com bancos de dados como MySQL para armazenar informações de forma organizada.
  • Em projetos pessoais, desenvolvi algumas aplicações web simples, integrando APIs básicas.
  • Na faculdade, participei de projetos em grupo, onde pude trabalhar com meus colegas, melhorando minhas habilidades de trabalho em equipe e comunicação.

Acredito que continuar aprendendo, praticando e enfrentando novos desafios é essencial para me tornar um desenvolvedor competente.
Tenho o objetivo de me tornar um profissional cada vez mais preparado para as demandas do mercado.
A cada novo projeto, busco aplicar o que aprendi e aprender ainda mais, sempre com a mesma paixão e curiosidade que me acompanham desde a adolescência.
